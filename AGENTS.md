# Repository Guidance

## Codex Workspace
Start Codex for this repository with `./scripts/codex-project`. The launcher uses the shared `CODEX_HOME=$HOME/.codex`, which points at the `gus-projects` Linear workspace.

For Linear work in this repository, use the `linear` MCP server with:
- team key `AIF`
- default project `AI Financial Planner MVP`

If a request is specific to this repo and no other Linear target is given, default to that team and project.
